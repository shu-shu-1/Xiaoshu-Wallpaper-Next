## 概述 ℹ️

小树壁纸 Next是一款在Windows平台上的壁纸软件

基于[小树壁纸](https://github.com/shu-shu-1/Xiaoshu-Wallpaper/)重构制作

这是一款多功能壁纸应用程序，旨在快速更换和下载来自多种来源的壁纸，包括 Bing、360 和 Wallhaven等。✨ 

允许用户收藏并自动更换喜欢的壁纸。

如果您喜欢这个项目，不妨点个 ⭐️吧！
